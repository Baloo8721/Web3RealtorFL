# Web3RealtorFL

## Overview
Web3RealtorFL is an innovative real estate agent matching platform for Florida, combining traditional real estate expertise with modern web technologies.

## Features
- Free agent matching service
- Web3 and traditional real estate integration
- Comprehensive client intake form

## Deployment
This project is deployed using GitHub Pages. To view the live site, visit: https://baloo8721.github.io/Web3RealtorFL/

## Local Development
1. Clone the repository
2. Open `index.html` in your browser

## Technologies
- HTML5
- CSS3
- Inter Font
- Google Forms Integration
